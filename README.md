# PersonalLibray em java
Um programa para gerenciar sua biblioteca pessoal.

## Funcionalidades
### Implementadas
- Cadastrar um livro
- Pesquisar livros que estão cadastrados
- Apagar um livro cadastrado
### Para implementar
- Editar um livro cadastrado
- Status de leitura
- Gerenciamento a partir de um usuário
- Pesquisa avançada
